## Overview

Railux Android is a modern, feature-rich media player application designed for Android devices. Built with AI-powered capabilities, it provides an exceptional viewing experience with intelligent recommendations and seamless playback.

## Features

### 🎬 **Core Playback**
- Support for multiple media formats
- Hardware-accelerated video decoding
- Picture-in-Picture (PiP) mode support
- Background playback
- Subtitle support (SRT, VTT, ASS/SSA)

### 🤖 **AI Integration**
- Intelligent content recommendations
- Smart subtitle matching
- Automated metadata fetching
- AI-enhanced playback optimization

### 📱 **User Experience**
- Material Design interface
- Dark and Light themes
- Gesture controls (swipe to seek, volume/brightness adjustment)
- Continue watching functionality
- Multiple server support (Emby, WebDAV, SMB)

### 🌐 **Multi-Source Support**
- Jellyfin media server integration
- Emby server support
- WebDAV streaming
- SMB/CIFS network shares
- Local file playback

### 🔧 **Advanced Features**
- Media casting support
- Sleep timer
- Playback speed control
- Audio track selection
- Chapter navigation
- Trakt sync support

## Download

### Latest Release
Download the latest APK from our [Releases](https://github.com/Railux-AI-Player/Railux-Android/releases) page.

### System Requirements
- **Android Version**: 5.0 (Lollipop) or higher
- **Architecture**: ARM64-v8a, ARMv7, x86, x86_64
- **Storage**: ~50MB for installation

### Installation

1. Download the APK file from the [Releases](https://github.com/Railux-AI-Player/Railux-Android/releases) page
2. Enable "Install from unknown sources" in your Android settings
3. Open the downloaded APK file
4. Follow the installation prompts

## Technology Stack

- **Language**: Kotlin
- **Architecture**: MVVM + Clean Architecture
- **UI Framework**: Jetpack Compose
- **Media Player**: Media3 ExoPlayer
- **DI Framework**: Koin
- **Networking**: OkHttp + Retrofit
- **Database**: Room
- **Image Loading**: Coil
